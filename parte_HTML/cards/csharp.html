C# (C Sharp)

C# é uma linguagem de programação orientada a objetos desenvolvida pela Microsoft. É muito utilizada no desenvolvimento de aplicações desktop, web, jogos e sistemas corporativos.

Nível da linguagem

C# é uma linguagem de alto nível, criada para oferecer produtividade, segurança e facilidade no desenvolvimento de aplicações modernas através da plataforma .NET.

Tipagem

C# é uma linguagem estaticamente tipada e fortemente tipada. Isso significa que o tipo das variáveis é definido durante a compilação, ajudando a evitar erros.

Tipos primitivos

int (inteiros)

Números inteiros.


int idade = 20;
Console.WriteLine(idade);

double (decimais)

Números com casas decimais.


double altura = 1.75;
Console.WriteLine(altura);

char (caractere)

Representa um único caractere.


char letra = 'A';
Console.WriteLine(letra);

bool (lógico)

Valores verdadeiro ou falso.


bool ativo = true;
Console.WriteLine(ativo);

string (texto)

Sequência de caracteres.


string nome = "Gabriel";
Console.WriteLine(nome);

Onde C# é mais usado

C# é amplamente utilizado em diversas áreas:

  • Desenvolvimento de jogos com Unity
  • Aplicações desktop para Windows
  • Back-end web com ASP.NET
  • Sistemas corporativos
  • Aplicações em nuvem utilizando .NET

Dúvidas comuns sobre C#

C# é difícil de aprender?

Não. Sua sintaxe é organizada e bastante intuitiva, sendo uma ótima opção para iniciantes.

C# é apenas para Windows?

Não. Atualmente, com o .NET, aplicações C# podem rodar em Windows, Linux e macOS.

C# ainda é usado em 2026?

Sim. Continua sendo uma das linguagens mais utilizadas para desenvolvimento corporativo e criação de jogos.

C# serve para backend?

Sim. O framework ASP.NET é uma das principais tecnologias para desenvolvimento de APIs e aplicações web.

Vale a pena aprender C#?

Sim. É uma linguagem moderna, muito valorizada no mercado e utilizada em diversas áreas da tecnologia.
