JavaScript

JavaScript é uma linguagem de programação usada principalmente para desenvolvimento web, tornando páginas interativas e dinâmicas.

Nível da linguagem

JavaScript é uma linguagem de alto nível, interpretada e muito flexível, executada principalmente no navegador e também no backend com Node.js.

Tipagem

JavaScript é dinamicamente tipado e de tipagem fraca. Isso significa que variáveis podem mudar de tipo durante a execução.

Tipos de dados

Number

Representa números inteiros e decimais.


let idade = 20;
let altura = 1.75;

console.log(idade);
console.log(altura);

String

Texto.


let nome = "Gabriel";
console.log(nome);

Boolean

Verdadeiro ou falso.


let ativo = true;
console.log(ativo);

Object

Estrutura de dados com chave e valor.


let pessoa = {
    nome: "Gabriel",
    idade: 20
};

console.log(pessoa);

Undefined / Null

Valores vazios ou não definidos.


let x;
let y = null;

console.log(x);
console.log(y);

Onde JavaScript é mais usado

  • Desenvolvimento web (frontend)
  • Aplicações interativas em sites
  • Backend com Node.js
  • APIs e servidores
  • Aplicações mobile (React Native)

Dúvidas comuns sobre JavaScript

JavaScript é a mesma coisa que Java?

Não. Apesar do nome parecido, são linguagens completamente diferentes.

JavaScript roda só no navegador?

Não. Com Node.js, ele também roda no backend.

JavaScript é difícil?

Ele é simples no começo, mas pode ficar complexo conforme avança.

JavaScript é obrigatório pra web?

Sim, para interatividade em sites modernos.

Vale a pena aprender JavaScript?

Sim, é uma das linguagens mais importantes da web.
